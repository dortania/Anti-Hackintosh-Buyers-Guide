# Intel CPU Support

Many Intel CPUs work in most versions of macOS, especially as they can usually be spoofed to support older or newer versions of macOS. The table below assumes that a vanilla kernel with no modifications is being used.

::: warning

Apple has not shipped any devices with the below CPUs, making them more difficult to boot macOS on.

* X79/LGA 2011 and X99/LGA 2011-V3
  * These systems commonly have issues booting and can be difficult to set up. Avoid using if you can.
* Atoms, Celerons, and Pentiums
  * The iGPUs in these CPUs rarely work in macOS, meaning that a dedicated GPU is needed.
  * These CPUs may need their CPUID value spoofed as well

:::
